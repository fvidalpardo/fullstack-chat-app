# Bienvenidos a Chatty

Esta es una simple aplicación para chatear con mensajes, construido con las últimas tecnologías full-stack!

## Características:

- Tecnologías: MERN + Socket.io + TailwindCSS + Daisy UI
- Con Autenticación y Autorización con JWT
- Mensajes en tiempo real con Socket.io
- Estado en vivo de los usuarios
- Manejo de estados globales con Zustand
- Manejo de errores y exepciones en el servidor y el cliente
- y mucho más!

### Configuración del archivo .env

```js
MONGODB_URI=...
PORT=5001
JWT_SECRET=...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...

NODE_ENV=development
```

### Para buildear la aplicación

```shell
npm run build
```

### Para correr la aplicación

```shell
npm start
```
